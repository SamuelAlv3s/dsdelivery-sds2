# :pizza: DS Delivery

Ds delivery é uma aplicação desenvolvida durante a 2ª semana do DevSuperior.

O aplicativo desenvolvido durante essa semana, foi uma página web para registro de pedidos e um app para ver pedidos cadastrados, iniciar navegação, confirmar entrega ou cancelar. 

## :computer: Backend
O backend foi desenvolvido utilizando Java e o framework Springboot. 

## :envelope: Frontend web
O frontend web da aplicação foi construído utilizando typescript e reactjs. 

## :iphone: Frontend mobile
O frontend mobile da aplicação foi construído utilizando typescript e react native.
